# TIDAL GraphQL API

TIDAL is a music streaming service offering high-fidelity audio. The API covers track catalog, artist pages, album details, playlist management, user library, music metadata, and high-fidelity audio quality information.

**Endpoint:** No native GraphQL endpoint — conceptual schema derived from TIDAL REST API
**Documentation:** https://developer.tidal.com/documentation/api-sdk/api-sdk-overview

- Reference: https://tidal-music.github.io/tidal-api-reference/
- Schema: https://github.com/api-evangelist/tidal
